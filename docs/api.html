API Reference¶

Summary of the main modules used to generate and write scenes.

Entrada y salida de QLC+¶

.. py:module:: scenegen.qlc_io

Utility helpers to read and write QLC+ workspace files.

.. py:function:: load_rig_from_qlc(path) :module: scenegen.qlc_io

Load a rig definition from a QLC+ workspace (.qxw).

.. py:function:: write_scenes_to_qlc(path, rig, scenes, output_path=None, create_show=False, show_name=’Generated Show’, show_step_ms=5000, create_flash_chaser=False, flash_chaser_name=’Flash White’, flash_step_ms=100, flash_total_ms=300000, create_primary_sweep=False, primary_sweep_name=’Blue Sweep’, primary_sweep_step_ms=500) :module: scenegen.qlc_io

Append generated scenes to a QLC+ workspace.

The original file is left untouched; by default the function writes to <stem>_generated.qxw unless output_path is provided.

Data models to describe a QLC+ rig.

.. py:class:: ChannelDef(index, name, channel_type=’generic’, default_value=None) :module: scenegen.rig

Describe a single channel inside a fixture.

.. py:class:: FixtureDef(fixture_id, name, manufacturer, model, mode, universe, address, channels=) :module: scenegen.rig

Describe a fixture within the rig.

.. py:property:: FixtureDef.channel_count :module: scenegen.rig :type: int

  Return the number of channels this fixture exposes.

.. py:class:: Rig(name, fixtures=) :module: scenegen.rig

Data model for a full rig ready for scene generation.

Schemas describing generated scenes.

.. py:class:: FixtureState(fixture_id, channel_values=) :module: scenegen.schema

Specification of a single generated scene.

.. py:class:: SceneSet(title, scenes=) :module: scenegen.schema

Grouped set of generated scenes for a single song or theme.

Motor de generación¶

.. py:module:: scenegen.generator

Scene generation pipeline driven by a language model or rule-based selector.

.. py:function:: generate_scenes_for_song(rig, song_description, scene_context=None, scene_contexts=None, catalog_path=None, palettes_path=None, fixture_categories_path=None, llm_client=None) :module: scenegen.generator

Generate a SceneSet using rule-based selection or the LLM fallback.

.. py:module:: scenegen.prompt

Prompt builder to guide the language model.

.. py:function:: build_prompt(rig, song_description) :module: scenegen.prompt

Create a text prompt describing the rig and desired output schema.

.. py:module:: scenegen.llm_client

Lightweight client to talk to a local language model (e.g., Ollama).

.. py:class:: LLMClient(base_url=’http://localhost:11434’, model=’phi3:mini’) :module: scenegen.llm_client

Encapsulates calls to the configured LLM endpoint.

.. py:method:: LLMClient.generate(prompt, max_tokens=1024) :module: scenegen.llm_client

  Send the prompt to the model and return the raw response JSON.

.. py:module:: scenegen.scene_mapper

Translate semantic scenes into SceneSpec instances ready for QLC+.

.. py:function:: load_palettes(path) :module: scenegen.scene_mapper

Load color palettes from JSON (root key: palettes).

.. py:function:: load_fixture_categories(path) :module: scenegen.scene_mapper

Load fixture categories (and aliases) that map names to focus buckets.

.. py:function:: apply_scene(scene, rig, palettes, fixture_categories) :module: scenegen.scene_mapper

Build a SceneSpec from a semantic scene and a rig definition.

.. py:module:: scenegen.scene_selector

Rule-based semantic scene selector inspired by the SynkroDMX engine.

.. py:class:: SceneContext(energy, last_palette=None, last_scene=None, is_drop=False, strobe_allowed=True, section=None, tempo=None) :module: scenegen.scene_selector

Music context and recent state used to pick the next scene.

.. py:class:: SemanticScene(name, energy, palette, motion, strobe, focus, meta=None) :module: scenegen.scene_selector

Semantic (non-DMX) scene with the model parameters.

.. py:method:: SemanticScene.from_dict(data) :module: scenegen.scene_selector :classmethod:

  Build a ``SemanticScene`` from a plain dictionary.

.. py:function:: load_scene_catalog(path) :module: scenegen.scene_selector

Load a semantic scene catalog from JSON (root key: scenes).

.. py:function:: select_scene(context, catalog) :module: scenegen.scene_selector

Select a scene applying the methodology filters.
